Dropdown List Example

To create a dropdown list:

  1. Select the cell for the dropdown
  2. Go to Data > Data Validation
  3. Set "Allow" to "List"
  4. Enter values in "Source" (e.g., "Red,Green,Blue")
  5. Click OK

Quick Example: Goal Seek

To determine what sales value is needed to reach a specific profit:

  1. Set up a profit formula (e.g., =Sales-Expenses)
  2. Go to Data > What-If Analysis > Goal Seek
  3. Set cell: Select the profit cell
  4. To value: Enter target profit (e.g., 50000)
  5. By changing cell: Select the sales cell
  6. Click OK to see what sales are needed
