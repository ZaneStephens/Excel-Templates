Excel Formula Reference Cheat Sheet

Note: This cheat sheet provides a quick reference for commonly used Excel formulas and functions. Functions are organized by category for easy reference.

Math & Trigonometry Functions

Function Syntax Description Example
SUM =SUM(number1, [number2], ...) Adds all the numbers in a range of cells =SUM(A1:A10)
AVERAGE =AVERAGE(number1, [number2], ...) Returns the average of the arguments =AVERAGE(B1:B20)
MIN =MIN(number1, [number2], ...) Returns the smallest number in a set of values =MIN(C1:C100)
MAX =MAX(number1, [number2], ...) Returns the largest number in a set of values =MAX(C1:C100)
ROUND =ROUND(number, num_digits) Rounds a number to a specified number of digits =ROUND(A1, 2)
ROUNDUP =ROUNDUP(number, num_digits) Rounds a number up to a specified number of digits =ROUNDUP(A1, 0)
ROUNDDOWN =ROUNDDOWN(number, num_digits) Rounds a number down to a specified number of digits =ROUNDDOWN(A1, 0)
PRODUCT =PRODUCT(number1, [number2], ...) Multiplies all the numbers given as arguments =PRODUCT(A1:A5)
POWER =POWER(number, power) Returns the result of a number raised to a power =POWER(5, 2)
SQRT =SQRT(number) Returns the square root of a number =SQRT(16)
ABS =ABS(number) Returns the absolute value of a number =ABS(-10)
MOD =MOD(number, divisor) Returns the remainder after a number is divided by a divisor =MOD(10, 3)

Lookup & Reference Functions

Function Syntax Description Example
VLOOKUP =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup]) Looks for a value in the leftmost column of a table and returns a value in the same row from a column you specify =VLOOKUP("Smith", A1:C20, 3, FALSE)
HLOOKUP =HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup]) Looks for a value in the top row of a table and returns a value in the same column from a row you specify =HLOOKUP("Sales", A1:G3, 3, FALSE)
XLOOKUP =XLOOKUP(lookup_value, lookup_array, return_array, [not_found], [match_mode], [search_mode]) Searches a range or array and returns a corresponding item from a second range or array =XLOOKUP(A2, B2:B10, C2:C10, "Not found")
INDEX =INDEX(array, row_num, [column_num]) Returns a value at a given position in a range or array =INDEX(A1:C10, 5, 3)
MATCH =MATCH(lookup_value, lookup_array, [match_type]) Returns the position of a value in a range or array =MATCH("Smith", A1:A20, 0)
INDEX+MATCH =INDEX(return_range, MATCH(lookup_value, lookup_range, 0)) Combines INDEX and MATCH for powerful lookups =INDEX(C1:C20, MATCH("Smith", A1:A20, 0))
OFFSET =OFFSET(reference, rows, cols, [height], [width]) Returns a reference offset from a given reference =OFFSET(A1, 2, 3, 1, 1)
INDIRECT =INDIRECT(ref_text, [a1]) Returns a reference specified by a text string =INDIRECT("A" & B1)

Text Functions

Function Syntax Description Example
CONCAT =CONCAT(text1, [text2], ...) Joins text strings without using the & operator =CONCAT("Hello ", "World")
TEXTJOIN =TEXTJOIN(delimiter, ignore_empty, text1, [text2], ...) Combines text from multiple ranges with a delimiter =TEXTJOIN(", ", TRUE, A1:A5)
LEFT =LEFT(text, [num_chars]) Returns the first characters from a text string =LEFT(A1, 3)
RIGHT =RIGHT(text, [num_chars]) Returns the last characters from a text string =RIGHT(A1, 4)
MID =MID(text, start_num, num_chars) Returns characters from the middle of a text string =MID(A1, 3, 5)
LEN =LEN(text) Returns the number of characters in a text string =LEN(A1)
FIND =FIND(find_text, within_text, [start_num]) Returns the position of one text string within another (case-sensitive) =FIND("o", A1, 1)
SEARCH =SEARCH(find_text, within_text, [start_num]) Returns the position of one text string within another (not case-sensitive) =SEARCH("o", A1, 1)
SUBSTITUTE =SUBSTITUTE(text, old_text, new_text, [instance_num]) Replaces old text with new text in a text string =SUBSTITUTE(A1, "World", "Excel")
TRIM =TRIM(text) Removes extra spaces from text =TRIM(" Hello World ")
UPPER =UPPER(text) Converts text to uppercase =UPPER(A1)
LOWER =LOWER(text) Converts text to lowercase =LOWER(A1)
PROPER =PROPER(text) Capitalizes the first letter in each word =PROPER("john smith")

Date & Time Functions

Function Syntax Description Example
TODAY =TODAY() Returns the current date =TODAY()
NOW =NOW() Returns the current date and time =NOW()
DATE =DATE(year, month, day) Returns the serial number of a date =DATE(2025, 3, 15)
DAY =DAY(serial_number) Returns the day of the month (1-31) =DAY(TODAY())
MONTH =MONTH(serial_number) Returns the month (1-12) =MONTH(TODAY())
YEAR =YEAR(serial_number) Returns the year =YEAR(TODAY())
WEEKDAY =WEEKDAY(serial_number, [return_type]) Returns the day of the week (1-7) =WEEKDAY(TODAY(), 1)
DATEDIF =DATEDIF(start_date, end_date, unit) Calculates difference between dates in various units (Y, M, D, etc.) =DATEDIF("1/1/2023", TODAY(), "M")
EDATE =EDATE(start_date, months) Returns date that is the indicated number of months before/after start date =EDATE(TODAY(), 3)
NETWORKDAYS =NETWORKDAYS(start_date, end_date, [holidays]) Returns the number of working days between two dates =NETWORKDAYS("3/1/2025", "3/31/2025")
WORKDAY =WORKDAY(start_date, days, [holidays]) Returns a date that is a specified number of working days in the future =WORKDAY(TODAY(), 10)

Financial Functions

Function Syntax Description Example
PMT =PMT(rate, nper, pv, [fv], [type]) Calculates the payment for a loan based on constant payments and interest rate =PMT(5%/12, 5*12, 10000)
FV =FV(rate, nper, pmt, [pv], [type]) Returns the future value of an investment =FV(6%/12, 10*12, -100, -1000)
PV =PV(rate, nper, pmt, [fv], [type]) Returns the present value of an investment =PV(8%/12, 20*12, -500)
RATE =RATE(nper, pmt, pv, [fv], [type], [guess]) Returns the interest rate per period of a loan or investment =RATE(36, -300, 10000)
NPER =NPER(rate, pmt, pv, [fv], [type]) Returns the number of periods for an investment =NPER(6%/12, -200, 10000)
IPMT =IPMT(rate, per, nper, pv, [fv], [type]) Returns the interest payment for a given period of an investment =IPMT(6%/12, 1, 36, 10000)
PPMT =PPMT(rate, per, nper, pv, [fv], [type]) Returns the principal payment for a given period of an investment =PPMT(6%/12, 1, 36, 10000)
IRR =IRR(values, [guess]) Returns the internal rate of return for a series of cash flows =IRR(A1:A10)
NPV =NPV(rate, value1, [value2], ...) Returns the net present value of an investment based on cash flows =NPV(10%, B1:B10)
Pro Tip: Use absolute cell references (with $ sign) in formulas when you want to prevent them from changing when copied to other cells. Example: =A$1 will keep row 1 fixed when copying down.

Array Functions (Excel 365)

Function Syntax Description Example
FILTER =FILTER(array, include, [if_empty]) Filters a range of data based on criteria you define =FILTER(A1:C20, B1:B20>100, "No results")
SORT =SORT(array, [sort_index], [sort_order], [by_col]) Sorts the contents of a range or array =SORT(A1:C20, 2, -1)
SORTBY =SORTBY(array, by_array1, [sort_order1], ...) Sorts array based on corresponding values in by_array =SORTBY(A1:C20, D1:D20, 1)
UNIQUE =UNIQUE(array, [by_col], [exactly_once]) Returns a list of unique values from a range or array =UNIQUE(A1:A20)
SEQUENCE =SEQUENCE(rows, [cols], [start], [step]) Generates a sequence of numbers =SEQUENCE(10, 1, 5, 5)
RANDARRAY =RANDARRAY([rows], [columns], [min], [max], [integer]) Returns an array of random numbers =RANDARRAY(5, 3, 1, 100, TRUE)
