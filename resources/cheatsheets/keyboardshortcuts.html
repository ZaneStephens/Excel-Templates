Excel Keyboard Shortcuts Cheat Sheet

Note: This cheat sheet contains the most commonly used keyboard shortcuts in Microsoft Excel. Memorizing these shortcuts will significantly improve your productivity and efficiency.

Navigation Shortcuts

Shortcut Description
Ctrl + Home Go to cell A1
Ctrl + End Go to the last used cell
Ctrl + Arrow Keys Move to the edge of current data region
Ctrl + Page Up / Page Down Move to previous/next worksheet
Alt + Page Up / Page Down Move one screen to the left/right
F5 Open Go To dialog box
Ctrl + G Open Go To dialog box
Ctrl + F Find
Ctrl + H Replace

Selection Shortcuts

Shortcut Description
Ctrl + Space Select entire column
Shift + Space Select entire row
Ctrl + A Select all data in the current region (press again for entire worksheet)
Ctrl + Shift + Arrow Keys Extend selection to the last cell with data in that direction
Shift + Arrow Keys Extend selection by one cell
Ctrl + Shift + * Select current region (data surrounded by blank cells)
F8 Toggle extend mode (use arrow keys to extend selection)
Shift + Click Extend selection to clicked cell
Ctrl + Click Add clicked cell/range to selection

Editing Shortcuts

Shortcut Description
F2 Edit active cell
Ctrl + Z Undo
Ctrl + Y Redo
Ctrl + X Cut
Ctrl + C Copy
Ctrl + V Paste
Ctrl + Alt + V Paste Special dialog
Delete Delete cell content
Ctrl + Delete Delete text to the end of the line
F4 Repeat last action or toggle absolute/relative references in formula
Ctrl + D Fill down (copy content of top cell to selection)
Ctrl + R Fill right (copy content of leftmost cell to selection)
Ctrl + ; Insert current date
Ctrl + Shift + : Insert current time

Formatting Shortcuts

Shortcut Description
Ctrl + B Bold
Ctrl + I Italic
Ctrl + U Underline
Ctrl + 1 Format Cells dialog
Ctrl + Shift + ~ Apply General format
Ctrl + Shift + $ Apply Currency format
Ctrl + Shift + % Apply Percentage format
Ctrl + Shift + # Apply Date format
Ctrl + Shift + @ Apply Time format
Alt + H + O + R Adjust row height
Alt + H + O + C Adjust column width
Alt + H + O + I AutoFit row height
Alt + H + O + A AutoFit column width

Formula Shortcuts

Shortcut Description
Alt + = AutoSum (SUM function)
Shift + F3 Insert function (Function Wizard)
Ctrl + ` (backtick) Toggle between showing formulas and their values
F4 (in formula) Toggle absolute/relative reference
F9 (in formula) Calculate all workbooks
Shift + F9 Calculate active worksheet
Ctrl + Alt + F9 Recalculate all workbooks
Ctrl + Shift + Enter Enter array formula
Alt + Down Arrow Display AutoComplete list/dropdown list

Data and PivotTable Shortcuts

Shortcut Description
Ctrl + Shift + L Toggle filter
Alt + Down Arrow Open filter dropdown on filtered column
Alt + A + S + S Sort A to Z
Alt + A + S + D Sort Z to A
Alt + A + T Create PivotTable
Alt + D + P Open Data Form
Alt + D + R Data Validation

File Operation Shortcuts

Shortcut Description
Ctrl + N New workbook
Ctrl + O Open workbook
Ctrl + S Save workbook
F12 Save As
Ctrl + W Close workbook
Ctrl + P Print dialog
Ctrl + F2 Print preview

View Shortcuts

Shortcut Description
Ctrl + F1 Show/hide ribbon
Alt + W + F + F Freeze/unfreeze panes
Alt + W + V + G Show/hide gridlines
Alt + W + V + H Show/hide headings
Alt + W + Q Zoom dialog
Alt + Tab Switch between open applications
Ctrl + Tab Switch between open workbooks
