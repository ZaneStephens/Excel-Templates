Excel Chart Selection Guide

Choosing the right chart type is crucial for effectively communicating your data. This guide will help you select the most appropriate chart type based on what you want to show.

Chart Selection Decision Tree

Start by asking yourself what you want to show with your data:

  • Comparison between items? → Bar/Column Chart, Radar Chart
  • Composition (parts of a whole)? → Pie Chart, Stacked Bar/Column, Area Chart
  • Distribution of data? → Histogram, Box Plot, Scatter Plot
  • Relationship between variables? → Scatter Plot, Bubble Chart, Line Chart
  • Trend over time? → Line Chart, Area Chart, Column Chart with trendline
  • Geographical data? → Map Chart

Comparison Charts

Use these charts when you want to compare values across different categories.

Column Chart

[Column Chart Image]

Best for:

  • Comparing values across categories
  • Showing changes over time (limited periods)
  • Displaying negative values
  • Comparing up to ~15 categories

Avoid for:

  • Too many categories (becomes cluttered)
  • Showing precise values over time (use line chart)
Tip: Sort columns by value (not alphabetically) to make patterns more visible.

Bar Chart

[Bar Chart Image]

Best for:

  • Comparing values across categories
  • Categories with long names
  • Displaying negative values
  • Comparing many categories (more than columns)

Avoid for:

  • Time series data (dates don't read well horizontally)
  • When vertical space is limited
Tip: Sort bars in ascending/descending order for better readability.

Radar Chart

[Radar Chart Image]

Best for:

  • Comparing multiple variables for multiple items
  • Performance evaluations across dimensions
  • Showing strengths/weaknesses patterns

Avoid for:

  • More than 2-3 items (becomes cluttered)
  • When precise comparisons are needed
  • When variables have different scales
Tip: Limit to 5-10 dimensions and normalize data to same scale.

Composition Charts

Use these charts when you want to show how parts make up a whole.

Pie Chart

[Pie Chart Image]

Best for:

  • Showing parts of a whole (percentages)
  • When you have few categories (2-6)
  • When one category is significantly different

Avoid for:

  • More than 6 categories
  • When categories have similar values
  • Comparing precise values
Tip: Add data labels with percentages and consider a donut chart for better labeling.

Stacked Column/Bar Chart

[Stacked Chart Image]

Best for:

  • Showing parts of a whole across categories
  • Comparing total values across categories
  • Showing changes in composition over time

Avoid for:

  • Too many stack segments (>5-7)
  • When comparing individual segments is important
Tip: Use 100% stacked charts to focus on percentage composition rather than absolute values.

Area Chart

[Area Chart Image]

Best for:

  • Showing composition changes over time
  • Emphasizing the total volume
  • Showing part-to-whole relationships over time

Avoid for:

  • More than 3-5 categories (becomes cluttered)
  • When precise individual values are important
  • When smaller categories are hidden behind larger ones
Tip: Consider using 100% stacked area charts to focus on proportional changes.

Distribution Charts

Use these charts when you want to show how data is distributed across a range.

Histogram

[Histogram Image]

Best for:

  • Showing distribution of continuous data
  • Identifying patterns, skewness, and outliers
  • Analyzing frequency distributions

Avoid for:

  • Categorical data
  • Small data sets
  • When exact values are important
Tip: Choose bin sizes carefully to reveal the true shape of your data distribution.

Box and Whisker Plot

[Box Plot Image]

Best for:

  • Showing data distribution summary
  • Comparing distributions across categories
  • Identifying outliers and data spread

Avoid for:

  • Small data sets
  • When audience is unfamiliar with box plots
  • When detailed distribution shape is important
Tip: Include a brief explanation of how to read a box plot for audiences unfamiliar with them.

Relationship Charts

Use these charts when you want to show how variables relate to each other.

Scatter Plot

[Scatter Plot Image]

Best for:

  • Showing relationship between two variables
  • Identifying correlations and patterns
  • Spotting outliers and clusters

Avoid for:

  • Time series data (use line charts)
  • Categorical data
  • When you have very few data points
Tip: Add a trendline to highlight the relationship direction and strength.

Bubble Chart

[Bubble Chart Image]

Best for:

  • Showing relationships between three variables
  • When size represents an important third dimension
  • Comparing items across multiple dimensions

Avoid for:

  • When precise size comparison is important
  • Too many data points (becomes cluttered)
  • When the third variable doesn't add meaningful context
Tip: Add data labels to important bubbles and include a size legend.

Trend Charts

Use these charts when you want to show how data changes over time.

Line Chart

[Line Chart Image]

Best for:

  • Showing trends over time
  • Comparing trends across multiple series
  • Continuous data with many data points

Avoid for:

  • Categorical comparisons (use column/bar)
  • Too many lines (>5-7 becomes cluttered)
  • Sparse data points
Tip: Use markers to highlight specific data points and consider using different line styles.

Sparklines

[Sparklines Image]

Best for:

  • Showing trends in a small space
  • Embedding trends within tables
  • Providing quick visual context

Avoid for:

  • When detailed analysis is needed
  • When exact values are important
  • When comparing multiple series directly
Tip: Highlight min/max points and use consistent scales when comparing multiple sparklines.

Specialized Charts

Use these charts for specific data visualization needs.

Map Chart

[Map Chart Image]

Best for:

  • Geographical data visualization
  • Regional comparisons
  • Showing spatial patterns

Avoid for:

  • Non-geographical data
  • When precise values are important
  • Small regions that are hard to see
Tip: Use color gradients effectively and consider adding data labels for important regions.

Waterfall Chart

[Waterfall Chart Image]

Best for:

  • Showing sequential additions and subtractions
  • Financial statements (P&L, cash flow)
  • Inventory changes

Avoid for:

  • Non-sequential data
  • When individual values are more important than cumulative effect
  • Too many steps (becomes hard to follow)
Tip: Use different colors for increases and decreases, and add a total column.

Combo Chart

[Combo Chart Image]

Best for:

  • Comparing different types of related data
  • Showing data with different scales
  • Highlighting relationships between different metrics

Avoid for:

  • Unrelated data series
  • More than 2-3 different chart types (becomes confusing)
  • When simplicity is important
Tip: Use a secondary axis when values have different scales and clearly label both axes.

Best Practices for Chart Design

  • Simplicity: Keep charts simple and focused on the main message
  • Consistency: Use consistent colors, styles, and scales across related charts
  • Labeling: Include clear titles, axis labels, and data labels where appropriate
  • Color: Use color purposefully to highlight important information
  • Scale: Use appropriate scales and consider starting y-axis at zero for bar/column charts
  • Sorting: Sort data in a meaningful way (not just alphabetically)
  • Clutter: Remove unnecessary gridlines, borders, and decorative elements
  • Context: Provide context with benchmarks, targets, or historical averages
